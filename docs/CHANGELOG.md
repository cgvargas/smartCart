# SmartCart - Changelog

## [1.0.0] - 2024-12-23

### Added
- Initial project structure (backend + mobile)
- Django REST API with JWT authentication
- Custom User model with email login
- Payment methods management (CRUD)
- Shopping lists with budget tracking
- Shopping items with subtotal calculation
- Product catalog with price history
- React Native mobile app with Expo
- Camera integration for product scanning
- OCR service (Tesseract.js)
- Budget progress bar with alerts
- History with list duplication feature
- Profile settings with alert percentage

### Backend Apps
- `accounts` - User management and authentication
- `payments` - Payment methods CRUD
- `shopping` - Shopping lists and items
- `products` - Product catalog and price history

### Mobile Screens
- Login / Register
- Home Dashboard
- Shopping List
- New List Creation
- Camera (OCR)
- Product Detail
- History
- Profile
- Payments
